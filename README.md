Homework API Automation using Mocha 
You Need install all of module to running this project 

1. Mocha JS -> https://mochajs.org/
2. Chai JS -> https://www.chaijs.com/
3. Node Fetch -> https://www.npmjs.com/package/node-fetch
4. AJV Validator -> https://www.npmjs.com/package/ajv
5. Mocha Simpel HTML Reporter -> https://www.npmjs.com/package/mocha-simple-html-reporter

If you have problem when running, DM Me!!! 
Script running -> npm run jalanin (custom script)
